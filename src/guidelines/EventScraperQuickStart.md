     - Tags

4. **Filter by Category**
   - Choose from:
     - Hackathon
     - Workshop
     - Competition
     - Project Expo
     - Symposium
     - Conference

5. **Filter by Mode**
   - Online
   - Offline
   - Hybrid

### Registering for Events

1. **Click "Register Now"** button on any event card
2. This redirects you to the event's official registration page
3. Complete registration on the event platform

### Sharing Events

1. Click the **Share** icon (three dots)
2. Choose platform:
   - Twitter
   - LinkedIn
   - WhatsApp
3. Share with friends or save for later

### Downloading Event Details

1. Click the **Download** icon
2. Event details saved as `.txt` file
3. Includes all information:
   - Event name and organizer
   - Dates and deadlines
   - Location and mode
   - Prizes and eligibility
   - Registration link
   - Tags and domains

### Applying for OD

Some events allow direct OD application:
1. Event details pre-filled
2. Modify if needed
3. Submit OD request
4. Track status in "My OD Requests" tab

## Frequently Asked Questions

### Q: How often are events updated?
**A:** Events are refreshed every 6 hours automatically. You can manually refresh anytime by clicking the "Refresh" button.

### Q: Can I filter events from specific platforms?
**A:** Yes! Use the "Platform" filter to see events from specific sources like Unstop, Devpost, etc.

### Q: What does the "Verified" badge mean?
**A:** Verified events are from official sources with confirmed authenticity. Registration links are validated.

### Q: Why are some events showing "TBD" for location?
**A:** Some event organizers haven't announced the venue yet. Check back later or visit the event website for updates.

### Q: Can I apply for OD directly from the event card?
**A:** Currently, this feature is available for select events. Click "Apply for OD" when available, and the form will be pre-filled with event details.

### Q: How accurate is the deadline urgency indicator?
**A:** The system calculates urgency in real-time based on today's date. Always double-check the actual deadline date on the event page.

### Q: What if I find duplicate events?
**A:** The system automatically removes duplicates. If you spot any, they likely have slight variations in details.

### Q: Can I suggest adding a new event platform?
**A:** Yes! Contact the ERP admin or your HOD with the platform name and URL.

### Q: Are the prize amounts accurate?
**A:** Prize amounts are extracted from event listings. Always verify on the official event page as they may change.

### Q: Can I see events from past months?
**A:** The system shows only upcoming and ongoing events. Past events are automatically hidden.

## Troubleshooting

### Issue: No events showing
**Solution:**
- Clear all filters
- Click "Refresh" button
- Check internet connection
- Try a different browser

### Issue: Registration link not working
**Solution:**
- Copy the link and paste in new tab
- Check if event registration is still open
- Visit the event website directly
- Contact event organizers

### Issue: Wrong domain showing
**Solution:**
- Events may belong to multiple domains
- Check all tags and description
- Use search to filter more precisely

### Issue: Can't download event details
**Solution:**
- Check browser download settings
- Allow pop-ups for this site
- Try a different browser
- Clear browser cache
